# Order of the Dawn

**Мотивационное приложение в стиле магической академии**  
Выполняй ритуалы -> получай алмазы -> прокачивай питомца -> исследуй магический мир

---

## Экраны

| Экран | Описание |
|-------|----------|
| **Питомец** | Выбор питомца, прокачка уровня и статов |
| **Ритуалы** | Список заданий с наградой в алмазах |
| **Магазин** | Покупка и применение магических фонов |
| **Пророчества** | Мудрые цитаты через API |
| **Единение** | 3 источника энергии через async/await |

---

## Технологии

`Swift` • `UIKit` • `UITableView` • `UICollectionView` • `URLSession` • `async/await` • `UserDefaults`
